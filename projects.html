My Projects

Welcome to my personal portfolio — a curated showcase of my projects, technical journey, and contributions. Here, you'll find the applications I've built using technologies like Django, React.js and Node.js along with hands-on work in backend development and API integration.

Each project is a testament to my passion for problem-solving and my dedication to delivering high-quality solutions. Each project represents not just the final product, but the process behind it — from debugging complex backend logic to implementing responsive UIs, and even learning to leverage AI tools like GitHub Copilot and Grok for faster development and problem-solving.

I’m continuously evolving as a developer, and this space reflects that growth. Whether it's a full-stack application, a mobile app, or a hackathon project, I’ve documented my thought process, tools used, and challenges faced in building each one.

This portfolio is more than just a collection — it's a living record of how I learn, build, and improve.
